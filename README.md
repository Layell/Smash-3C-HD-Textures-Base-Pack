This is the base HD texture pack for the Smash 3C Project, and contains the base HD textures for every build.

As Smash 3C has a huge amount of HD textures, this allows for easy support and access for every build.

For textures specific to a build, be sure to download that specific build and its HD textures, and combine the folders:

Pplus: https://github.com/Layell/Smash-3C-Revised-Plus-Build

PM: https://github.com/Layell/Smash-3C-Revised-PM-Build

Brawl: https://github.com/Layell/Smash-3C-Revised-Brawl-Build

Minus: https://github.com/Layell/Smash-3C-Revised-Minus-Build

Alt Universe: https://github.com/Layell/Smash-3C-Alt-Universe-Patch
